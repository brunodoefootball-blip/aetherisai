# Aetheris AI - Arquitetura Técnica & Roadmap

## 1. Arquitetura Técnica
O Aetheris AI utiliza uma arquitetura **Full-Stack Moderna** otimizada para performance e escalabilidade.

### Stack Tecnológica:
- **Frontend**: React 19 + Tailwind CSS 4 (Design System Atômico).
- **Animações**: Motion (Framer Motion) para transições fluidas e interações de estado.
- **3D Engine**: Three.js + React Three Fiber para o fundo imersivo e elementos flutuantes.
- **Backend**: Express.js (Node.js) servindo como API e orquestrador de IA.
- **Banco de Dados**: SQLite (via better-sqlite3) para persistência leve e rápida de créditos e projetos.
- **IA**: Google Gemini 3 Flash para geração de código, copy e decisões estruturais em tempo real.

### Fluxo de Geração:
1. **Input**: O usuário fornece um prompt natural.
2. **Brain**: A IA processa o prompt e retorna um JSON estruturado (Design, Componentes, Endpoints).
3. **Synthesis**: O motor Aetheris converte o JSON em arquivos reais (React + Express).
4. **Deploy**: O sistema empacota o código e disponibiliza para exportação ou deploy instantâneo.

## 3. Roadmap de Crescimento (Startup de Alto Valor)

### Fase 1: Fundação (Meses 1-3)
- Lançamento da Beta Privada.
- Foco em Landing Pages de alta conversão.
- Integração com Google Analytics e Meta Pixel automática.

### Fase 2: Ecossistema (Meses 4-8)
- Lançamento do módulo de E-commerce.

### Fase 3: Expansão IA (Meses 9-12)
- Geração de vídeos curtos para anúncios baseados no site gerado.
- A/B Testing automático gerido por IA.
- Rodada de Investimento Seed (Valuation alvo: $5M - $10M).

### Fase 4: Dominação (Ano 2+)
- Marketplace de templates gerados por usuários.
- API Aetheris para integração em fluxos de trabalho enterprise.
